# 文件读取的结束
**读取如果是一行行的读取，一定要注意：每一行都有一个换行符'\n'，一定要strip掉，否则后续使用一定会出问题**
- 可以用一个for循环  
    ```python
    f = open('a.txt','r', encoding='utf-8')
    for line in f.readlines():
        line = line.strip('\n')
    ```
- 可以用while 循环
    ```python
    f = open('a.txt','r',encoding='utf-8')
    line = f.readline()
    while line:
        line = line.strip('\n')
        ...
        line = f.readline()
    ```
